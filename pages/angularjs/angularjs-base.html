Setting up

To create a module
angular.module('module-name', [])
Some available options after creating a module are:
  • constant: Defines a named, injectable constant value.
  • config: Only providers and constant are available at this point..
  • run: Instances can be injected here, but not providers.
  • factory: Instances can be injected here, but not providers.
  • controller: Creates a controller
An example from angularjs documentation
angular.module('myModule', []). .value('a', 123) .factory('a', function() { return 123; }) .directive('directiveName', ...) .filter('filterName', ...);
